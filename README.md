## CCNA v7 Module 01 - Network Design and Configuration

## 📌 Project Overview
This project focuses on designing and configuring a multi-router network using Cisco Packet Tracer. It demonstrates key networking concepts including static routing, VLSM subnetting, and basic network security.

## 🛠️ Technologies Used
- Cisco Packet Tracer  
- Cisco IOS CLI  

## 🚀 Features
- Multi-router network configuration  
- Static routing implementation  
- VLSM subnetting  
- Basic network security configuration  
- End-to-end connectivity testing  

## 🎯 Learning Objectives
- Understand network design fundamentals  
- Configure routers using CLI  
- Implement static routing  
- Apply VLSM subnetting  
- Configure basic security features
